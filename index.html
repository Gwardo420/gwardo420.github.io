Discord Commands

t!twitchsearch - Search's a Twitch Channel name. Example: t!twitchsearch Shroud

t!bal - Show's your BudCoin's balance! Example: t!bal

t!cash - Give's you 1,000 BudCoin's to start off. Example: t!cash

t!coinflip - Flip's a coin for a bet. Do you have the guts? Example: t!coinflip heads 100

t!stats - Show's your Twitch Buds server stats! Example: t!stats

t!twitch - Explain's how to setup your Twitch Account to appear in the LIVE-NOW channel. Example: t!twitch

Use these commands in the #bot-commands-rank channel in the Twitch Buds Discord! Please be considerate when using these commands as some of them may get spammy with all the users in the Discord simultaneously using them. Thank you for understanding & have fun!
